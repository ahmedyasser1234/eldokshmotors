# 🏎️ Eldoksh Motors - Luxury Car Rental & Sales Platform | منصة الدوكش للسيارات الفاخرة

A premium, full-stack digital solution designed for high-end vehicle rental and sales. This platform provides a seamless experience for both customers and administrators with real-time updates and secure payment processing.

---

## 🏗️ Technology Stack | التقنيات المستخدمة

### Frontend (الواجهة الأمامية)
- **React 19 & Vite 8**: For a lightning-fast user experience.
- **TypeScript**: Ensuring robust and type-safe code.
- **Tailwind CSS**: Modern utility-first styling.
- **Framer Motion**: State-of-the-art animations and transitions.
- **i18next**: Full multi-language support (Arabic/English) with RTL layout optimization.
- **Zustand & React Query**: Efficient state management and data fetching.

### Backend (الواجهة الخلفية)
- **NestJS 11**: Progressively scalable Node.js framework.
- **PostgreSQL**: Reliable relational database powered by **TypeORM**.
- **Redis**: High-performance caching and messaging.
- **Socket.io**: Real-time communication for instant notifications.
- **Stripe**: Secure PCI-compliant payment processing.
 Monovore management with multi-service support.

---

## ✨ Key Features | المميزات الرئيسية

- **Premium Aesthetics**: Featuring organic SVG wave transitions and premium dark-mode elements.
- **Fleet Showcase**: High-resolution galleries for "Recently Added" and "Featured Sales".
- **Real-time Notifications**: Instant alerts for new messages, sales requests, and payment updates.
- **Mobile-First Design**: Fully responsive UI/UX tailored for luxury enthusiasts on the go.
- **Secure Checkout**: Seamless Stripe integration for rentals and sales transactions.
- **Admin Command Center**: Complete control over vehicle inventory, reviews, and transaction history.

---

## 🚀 Deployment | الرفع والتشغيل

- **Frontend Hosting**: Deployed on [Netlify](https://eldokshmators.netlify.app).
- **Backend Hosting**: Self-hosted on a **Hostinger VPS** running on port **4321**.
- **Database**: Production-ready PostgreSQL instance.

## 🛠️ Installation | التثبيت والتشغيل المحلي

### 1. Clone the repository
```bash
git clone https://github.com/ahmedyasser1234/eldokshmotors.git
cd eldokshmotors
```

### 2. Backend Setup
```bash
cd backend
npm install
npm run start:dev
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
